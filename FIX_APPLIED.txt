                  КАК ОБНОВИТЬ РАСШИРЕНИЕ
═══════════════════════════════════════════════════════════════════

1️⃣  Удалите старую версию (если установлена)

   VSCode → Extensions (Ctrl+Shift+X)
   → Найдите "Claude Commit"
   → Шестерёнка → Uninstall

   ИЛИ через командную строку:
   
   code --uninstall-extension your-publisher-name.claude-commit

2️⃣  Установите новую версию

   Способ А: Через UI
   
   VSCode → Extensions (Ctrl+Shift+X)
   → ... (три точки) → Install from VSIX
   → Выберите: claude-commit-0.2.0.vsix
   
   Способ Б: Через командную строку
   
   code --install-extension claude-commit-0.2.0.vsix

3️⃣  Перезапустите VSCode

   Полностью закройте и откройте VSCode заново

4️⃣  Проверьте установку

   Command Palette (Ctrl+Shift+P)
   → Введите "Generate Commit"
   → Должна появиться команда: "Generate Commit Message with Claude"

✅ Добавлен activation event: "onStartupFinished"

   Теперь расширение активируется при запуске VSCode,
   а не только когда вызывается команда.

✅ Команда будет доступна сразу после запуска VSCode

═══════════════════════════════════════════════════════════════════
                    ПРОВЕРКА РАБОТЫ
═══════════════════════════════════════════════════════════════════

1. Откройте любой git репозиторий в VSCode

2. Сделайте изменения:
   
   echo "test" > test.txt
   git add test.txt

3. Откройте Source Control (Ctrl+Shift+G)

4. Нажмите ✨ (sparkle icon) в тулбаре

5. Если настроен Bedrock - должно сработать!
   Если настроен Anthropic - попросит API ключ

═══════════════════════════════════════════════════════════════════
                  ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════════

1. Проверьте логи расширения:

   Output Panel (Ctrl+Shift+U)
   → Выберите "Extension Host" в dropdown
   → Ищите строку: "Claude Commit extension is now active"

2. Проверьте, что расширение установлено:

   Extensions (Ctrl+Shift+X)
   → Ищите "Claude Commit Message Generator"
   → Должно быть включено (галочка)

3. Попробуйте команду через Command Palette:

   Ctrl+Shift+P
   → "Generate Commit Message with Claude"
   
   Если команда не найдена - расширение не активировалось

4. Проверьте консоль разработчика:

   Help → Toggle Developer Tools
   → Console tab
   → Ищите ошибки
