═══════════════════════════════════════════════════════════════════
  Вариант 1: AWS Bedrock (БЕЗ API ключа!)
═══════════════════════════════════════════════════════════════════

1. Настройте AWS credentials:
   
   aws configure
   
   Или используйте существующий AWS profile

2. Откройте VSCode Settings (Ctrl+,)

3. Найдите "Claude Commit" и настройте:
   
   Provider:    bedrock
   Aws Region:  us-east-1 (или ваш регион)
   Aws Profile: (оставьте пустым для default)

4. Готово! Используйте как обычно (✨ в Source Control)

═══════════════════════════════════════════════════════════════════
  Вариант 2: Anthropic API (требует ключ)
═══════════════════════════════════════════════════════════════════

1. Получите API ключ: https://console.anthropic.com/

2. В настройках:
   
   Provider: anthropic
   Api Key:  sk-ant-...

3. Готово!

═══════════════════════════════════════════════════════════════════

⚙️ НОВЫЕ НАСТРОЙКИ:

  claudeCommit.provider       - "anthropic" или "bedrock"
  claudeCommit.awsRegion      - AWS регион (для Bedrock)
  claudeCommit.awsProfile     - AWS профиль (опционально)
  
  Старые настройки работают как прежде!

🐛 TROUBLESHOOTING:

  "AccessDeniedException"
    → Добавьте IAM permission для bedrock:InvokeModel
  
  "Model not found"
    → Включите модель в Bedrock Console → Model Access
  
  "Unable to locate credentials"
    → Выполните: aws configure
  
  Больше решений в BEDROCK_SETUP.md!
