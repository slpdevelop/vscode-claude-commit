🚀 Как запустить:

  1. Откройте проект в VSCode:
     code E:\work\vscode-workspaces\vscode-claude-commit

  2. Нажмите F5 для запуска в режиме разработки

  3. В новом окне выберите провайдер:

     🅰️ Anthropic API:
        Settings → Claude Commit → Provider = "anthropic"
        Settings → Claude Commit → Api Key = "sk-ant-..."
        (Получить: https://console.anthropic.com/)

     🅱️ AWS Bedrock (БЕЗ API ключа!):
        aws configure  # настройте AWS credentials
        Settings → Claude Commit → Provider = "bedrock"
        Settings → Claude Commit → Aws Region = "us-east-1"
        (См. BEDROCK_SETUP.md для деталей)

  4. Создайте тестовый git repo и попробуйте:
     - Ctrl+Shift+G (Source Control)
     - Нажмите ✨ (sparkle icon)
     - Profit!

⚙️ Настройки (в VSCode Settings):

  🆕 claudeCommit.provider             - Провайдер: "anthropic" или "bedrock"
  claudeCommit.apiKey                  - API ключ Anthropic (для anthropic)
  🆕 claudeCommit.awsRegion            - AWS регион (для bedrock)
  🆕 claudeCommit.awsProfile           - AWS профиль (для bedrock)
  claudeCommit.model                   - Модель (Haiku/Sonnet/Opus)
  claudeCommit.useConventionalCommits  - Conventional Commits формат
  claudeCommit.analyzeAllChanges       - Все vs staged changes
  claudeCommit.maxDiffSize             - Макс размер diff
