🚀 Как запустить:

  1. Откройте проект в VSCode:
     code E:\work\vscode-workspaces\vscode-claude-commit

  2. Нажмите F5 для запуска в режиме разработки

  3. В новом окне настройте API ключ:
     Settings → Claude Commit → Api Key
     (Получить: https://console.anthropic.com/)

  4. Создайте тестовый git repo и попробуйте:
     - Ctrl+Shift+G (Source Control)
     - Нажмите ✨ (sparkle icon)
     - Profit!

⚙️ Настройки (в VSCode Settings):

  claudeCommit.apiKey                    - API ключ Anthropic
  claudeCommit.model                     - Модель (Haiku/Sonnet/Opus)
  claudeCommit.useConventionalCommits    - Conventional Commits формат
  claudeCommit.analyzeAllChanges         - Все vs staged changes
  claudeCommit.maxDiffSize               - Макс размер diff
