ПОПРОБУЙТЕ В ТАКОМ ПОРЯДКЕ:

═══════════════════════════════════════════════════════════════════
  ВАРИАНТ 1: Полная переустановка (30 секунд)
═══════════════════════════════════════════════════════════════════

1. Удалите старое расширение:
   
   VSCode → Extensions (Ctrl+Shift+X)
   → Найдите "Claude Commit"
   → Шестерёнка → Uninstall

2. Закройте VSCode ПОЛНОСТЬЮ (все окна!)

3. Установите заново:
   
   code --install-extension claude-commit-0.2.0.vsix

4. Откройте VSCode

5. Проверьте:
   
   Ctrl+Shift+P → "Generate Commit"
   
   Команда должна появиться!

═══════════════════════════════════════════════════════════════════
  ВАРИАНТ 2: Режим разработки (ВСЕГДА работает!)
═══════════════════════════════════════════════════════════════════

Если Вариант 1 не помог:

1. Откройте проект расширения:
   
   code E:\work\vscode-workspaces\vscode-claude-commit

2. Нажмите F5

3. В НОВОМ окне "[Extension Development Host]":
   
   - Откройте любой git репозиторий
   - Ctrl+Shift+G (Source Control)
   - Нажмите ✨ (sparkle icon)
   
   ИЛИ
   
   - Ctrl+Shift+P → "Generate Commit Message with Claude"

4. Это должно работать на 100%!

═══════════════════════════════════════════════════════════════════
  ВАРИАНТ 3: Проверка активации
═══════════════════════════════════════════════════════════════════

Проверьте, активировалось ли расширение:

1. Ctrl+Shift+U (Output panel)

2. В dropdown выберите "Extension Host"

3. Ищите строку:
   
   "Claude Commit extension is now active"

❌ Если НЕТ этой строки - расширение НЕ активировалось!

✅ Если ЕСТЬ - расширение работает, но команда не регистрируется

═══════════════════════════════════════════════════════════════════
  ДИАГНОСТИКА: Почему команда не найдена?
═══════════════════════════════════════════════════════════════════

Возможные причины:

1. Расширение не установлено
   
   Проверка:
   code --list-extensions | grep claude
   
   Должно вывести что-то с "claude-commit"

2. Расширение не активировалось
   
   Проверка:
   Ctrl+Shift+U → Extension Host → ищите "is now active"

3. VSCode не перезапущен после установки
   
   Решение:
   Закройте ВСЕ окна VSCode и откройте заново

4. Конфликт с другим расширением
   
   Решение:
   Отключите все расширения, оставьте только claude-commit

5. Кэш VSCode
   
   Решение:
   Закройте VSCode
   rmdir /s /q "%APPDATA%\Code\CachedExtensions"
   Откройте VSCode
